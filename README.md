# Dictionary App 📖

A simple and responsive Dictionary App built using HTML, CSS, and JavaScript. This application allows users to search for word meanings, pronunciations, and other language-related information through an easy-to-use interface.

## Features

* Search for word definitions
* Display word meanings and parts of speech
* Responsive and user-friendly design
* Fast and interactive interface
* Built with pure HTML, CSS, and JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Use

1. Enter a word in the search box.
2. Click the search button or press Enter.
3. View the meaning and details of the searched word.

## Project Structure

* `index.html` – Main structure of the application
* `style.css` – Styling and layout
* `script.js` – Application logic and functionality
